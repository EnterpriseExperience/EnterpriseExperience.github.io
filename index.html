Introduction

Welcome to the documentation for the Rayfield-inspired UI library! This page provides all the information you need to get started and use this library effectively.

Examples

Below is an example of how to use the library:


-- Create a new UI window
local Window = Rayfield:CreateWindow({
    Name = "My First Window",
    LoadingTitle = "Loading Example",
    LoadingSubtitle = "Please wait...",
    ConfigurationSaving = true
})

-- Add a button
Window:CreateButton({
    Name = "Click Me",
    Callback = function()
        print("Button clicked!")
    end
})

API

Here's a quick overview of the API functions:


-- Create a new Window
Rayfield:CreateWindow({
    Name = string,
    LoadingTitle = string,
    LoadingSubtitle = string,
    ConfigurationSaving = boolean
})

-- Create a Button
Window:CreateButton({
    Name = string,
    Callback = function()
        -- code here
    end
})
